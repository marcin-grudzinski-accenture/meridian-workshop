### Halcyon Industrial Supply — Operations Dashboard Remediation
*Vue 3 / FastAPI / PostgreSQL · 2025*

Halcyon engaged us to audit and remediate a dashboard delivered by a prior vendor that had known defects and no test coverage — a situation directly analogous to Meridian's. We conducted an independent defect audit, resolved 11 in-scope issues (filters, data inconsistencies, i18n gaps), and established a Playwright end-to-end test suite covering eight critical flows. The engagement was delivered fixed-fee in 10 weeks. Halcyon's IT team approved the first post-engagement release within two weeks of handover.

**Relevance:** Near-identical to R1 and R3. Same stack, same prior-vendor scenario, same IT gatekeeper dynamic.

### Vantex Logistics — Procurement Recommendations Engine
*Python / React · 2024*

Vantex needed a net-new capability surfacing purchase recommendations from stock levels, supplier lead times, and budget constraints — the same core problem as Meridian's R2. We designed and built the recommendation engine (backend logic + frontend view), integrated it into the existing application without disrupting live operations, and delivered mid-engagement to the operations team for feedback before final release.

**Relevance:** Directly comparable to R2. Budget-ceiling logic, stock-and-demand inputs, operations team as primary user.

### Meridian-class Mid-Market Distributor (NDA) — Architecture Documentation
*Vue 3 / FastAPI · 2025*

**Relevance:** Directly comparable to R4. Same stack, same "previous vendor left minimal docs" scenario.
